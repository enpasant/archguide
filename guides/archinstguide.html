Guia de Instalação do Arch Linux com archinstall

1. Preparação Inicial

Certifique-se de que você possui:

  • Uma mídia inicializável do Arch Linux (pen drive ou CD).
  • Conexão à internet.
Boot pelo pendrive e selecione a opção de inicialização padrão.

2. Verificar Conexão com a Internet

Teste a conexão utilizando:

ping archlinux.org

Se não houver conexão, configure-a com iwctl para redes Wi-Fi ou verifique o cabo Ethernet.

3. Iniciar o archinstall

O Arch Linux oferece um instalador automatizado. Para iniciá-lo, basta digitar:

archinstall

Siga as etapas do assistente:

  • Selecione o idioma e o layout do teclado.
  • Escolha o disco para instalação.
  • Configure a partição (o particionamento automático é recomendado para iniciantes).
  • Selecione o sistema de arquivos desejado (como ext4).
  • Defina o mirror para o repositório (opção automática é recomendada).
  • Configure o nome do usuário, senha e senha de root.
  • Escolha um ambiente gráfico, se necessário (GNOME, KDE, etc.).

4. Instalação e Configuração Final

Após concluir o assistente, o sistema será instalado. Aguarde até que o processo termine.

Reinicie o sistema utilizando o comando:
reboot

Remova a mídia de instalação para evitar que o sistema inicialize pelo pendrive novamente.

Aviso Importante

O Arch Linux é um sistema voltado para usuários avançados. Certifique-se de compreender cada etapa antes de continuar.
