Desktop Environments

A desktop environment (DE) is a great way to get started in Arch Linux.
To install one, please read this page.

1. - Installing

To install a desktop environment, you need internet access, and a compatible x-server.

1.2 - Installing a desktop environment

In Arch Linux, installing packages is done via pacman.

These are the commands necessary for installing;

KDE Plasma


sudo pacman -S plasma

This installs KDE Plasma.


xfce4


sudo pacman -S xfce4 xfce4-goodies

This installs xfce4 along with it's applications.


GNOME


sudo pacman -S gnome

This installs GNOME.


1.3 - Installing a display manager (DM)

You will need a display manager for acessing the chosen desktop environment.
To install one, see below.

LightDM

Download

GDM (Gnome Display Manager)

Download

SDDM (Simple Desktop Display Manager)

Download

LightDM


sudo pacman -S lightdm && sudo systemctl enable --now lightdm

This installs LightDM.


GDM (Gnome Display Manager)


sudo pacman -S gdm && sudo systemctl enable --now gdm

This installs GDM.


SDDM (Simple Desktop Display Manager)


sudo pacman -S sddm && sudo systemctl enable --now sddm

This installs SDDM.


Desktop Environments are a great way to start a fresh install from Arch Linux, make sure to do the best out of your system.
