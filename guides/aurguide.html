Guia Completo para o AUR (Arch User Repository)

O que é o AUR?

O AUR (*Arch User Repository*) é um repositório comunitário gerenciado por usuários do Arch Linux. Ele contém pacotes que não estão disponíveis nos repositórios oficiais, permitindo acesso a softwares adicionais, scripts e ferramentas criados pela comunidade.

Apesar de poderoso, é importante lembrar que os pacotes do AUR não são verificados oficialmente, então use-os com cautela.

Configurando o AUR

Para acessar e gerenciar pacotes do AUR, você precisará de um ajudante de AUR, como o yay ou o paru. Esses ajudantes facilitam a instalação, atualização e remoção de pacotes.

Instalando o yay (Ajudante do AUR)

Para instalar o yay, siga os passos abaixo:

  1. Certifique-se de ter o git instalado:
    2. sudo pacman -S git
  2. Clone o repositório do yay:
    3. git clone https://aur.archlinux.org/yay.git
  3. Entre no diretório do projeto clonado:
    4. cd yay
  4. Construa e instale o pacote:
    5. makepkg -si

Após a instalação, você poderá usar o yay para gerenciar pacotes do AUR.

Gerenciando Pacotes do AUR

Instalar um pacote

Para instalar um pacote do AUR, basta usar:

yay -S nome_do_pacote

Exemplo: Para instalar o Google Chrome:

yay -S google-chrome

Atualizar pacotes do AUR

Você pode atualizar todos os pacotes do sistema (incluindo os do AUR) com:

yay -Syu

Isso é equivalente ao comando sudo pacman -Syu, mas também atualiza pacotes do AUR.

Remover pacotes

Para remover um pacote do AUR, utilize:

yay -R nome_do_pacote

Para remover o pacote e suas dependências não utilizadas:

yay -Rns nome_do_pacote

Pesquisar pacotes

Você pode procurar por pacotes no AUR com:

yay -Ss palavra-chave

Exemplo: Para buscar pacotes relacionados ao VLC:

yay -Ss vlc

Precauções ao Usar o AUR

O AUR é uma ferramenta poderosa, mas requer cuidado. Algumas dicas para garantir a segurança ao usar o AUR:

  • Leia o código do arquivo PKGBUILD antes de instalar um pacote.
  • Evite instalar pacotes desconhecidos ou de fontes duvidosas.
  • Mantenha seus pacotes atualizados para evitar vulnerabilidades.
